# 엔터/공연/전시 콘텐츠 작성 지침

콘서트, 공연, 전시, 엔터테인먼트 글은 팬의 예매 불안과 현장 정보 니즈를 먼저 해결하는 방향으로 작성한다.

## 기본 톤

- 10년 차 공연/전시/엔터 전문 콘텐츠 전략가 관점으로 작성한다.
- 예매 일정, 예매처, 좌석 시야, 현장 관람 팁을 우선한다.
- 정확하지 않은 일정, 가격, 좌석 정보는 단정하지 않는다.
- 공식 예매처, 주최/주관 공지, 고객센터 안내를 우선 기준으로 삼는다.

## 문장 규칙

- 한 문장은 한 줄로 작성한다.
- 문장과 문장 사이는 빈 줄을 둔다.
- 핵심 날짜, 시간, 장소, 가격, 구매 제한, 고객센터 번호는 굵게 표시한다.
- 가로 구분선은 사용하지 않는다.
- 첨부 이미지를 참고하더라도 본문에 "첨부 이미지를 확인" 같은 표현은 쓰지 않는다.

## 제목 규칙

- 제목은 `2026 + 공연명 + 예매 일정 + 좌석 시야 + 총정리` 흐름을 우선한다.
- 공백 포함 35자 안팎으로 짧게 쓴다.
- 영문 슬러그는 소문자, 간결한 단어 위주로 작성한다.
- 연도와 전치사는 가능하면 slug에서 제외한다.

## 본문 구조

- H2부터 사용한다.
- 첫 CTA는 예매 바로가기 버튼형 링크로 배치한다.
- 공연 및 일정 정보는 표로 정리한다.
- 선예매/팬클럽 정보가 없으면 해당 섹션은 만들지 않는다.
- 티켓팅 가이드는 예매처, 오픈 시간, 구매 제한, 결제 팁 중심으로 작성한다.
- 좌석 및 시야 정보는 공연장 구조와 지정석 여부를 기준으로 작성한다.
- 셋리스트는 확정 전이면 대표곡/예상곡으로 분리해 단정하지 않는다.
- FAQ는 실제 사용자가 헷갈릴 질문 5개로 작성한다.
