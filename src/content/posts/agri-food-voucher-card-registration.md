농식품바우처 전용 카드를 받으셨나요?

이제 카드를 활성화하고 사용하는 방법이 궁금하실 텐데요.

카드는 수령 후 다음날 자동으로 활성화되며, 즉시 사용을 원하시면 고객센터 ARS로 간편하게 등록할 수 있습니다.

매월 1일 자동으로 포인트가 충전되어 전국 농식품 지정 매에서 일반 카드처럼 쉽게 사용 가능합니다.

단, 사용 기간은 매달 말일까지이며 당월 지원금액의 10% 이상 미사용 시 소멸되니 주의하세요.

👉 농식품바우처 사용처
## ⚡ 카드 등록 (활성화)

### 자동 활성화

**농식품 바우처카드 본인 수령 시 다음날 자동으로 활성화**

### 즉시 활성화 (ARS 등록)

**즉시 사용을 원하는 경우**

※ 즉시 사용을 원하는 경우 고객지원센터 **ARS(1551-0857**)를 통해 카드 사용등록이 가능합니다.

## 💰 포인트 충전

### 자동 충전 시스템

**매월 1일 자동 충전**

-   충전 일시: 매월 1일 오전
-   충전 방식: 자동 충전
-   별도 신청 불필요

**충전 금액**

-   가구원 수에 따라 차등 지급
-   1인 가구: 40,000원
-   10인 이상 가구: 187,000원

**충전 확인**

-   SMS 문자 알림
-   ARS 전화 조회 (1551-0857)
-   지정 매 결제 시 잔액 확인

### 포인트 이월 규정

**매달 1일 ~ 당월 말일까지 사용**

※ 사용하지 않은 금액은 이월되지 않으므로 반드시 해당 월 기간내에 사용해주시기 바랍니다. (매월 지원금액의 10% 미만에 한해서 이월 가능)

## 🛒 바우처 사용방법

### 사용 기간

**매달 1일 ~ 당월 말일까지**

※ 사용하지 않은 금액은 이월되지 않으므로 반드시 해당 월 기간내에 사용해주시기 바랍니다. (매월 지원금액의 10% 미만에 한해서 이월 가능)

### 오프라인 매장 사용

**일반 카드 결제와 동일한 방식으로 간편하게 사용 가능**

### 온라인 사용

**이용방법**

1.  **농식품바우처 홈페이지 로그인**
2.  **온라인 사용처 배너클릭**
3.  **온라인 사용처 이동 및 자동 로그인**
4.  **‘농식품바우처관’ 혹은 ‘농식품바우처 상품’ 상품 구입**

### 전입 시 주의사항

**농식품바우처 미시행 지자체로 전입시 바우처 사용이 불가하오니, 전입 전까지 지원금액을 모두 사용해야합니다.**

**미시행 지역**

-   서울: 강서구, 송파구 (청년 포함가구 미지원)
-   인천: 미추홀구
-   경기: 수원시, 부천시, 안산시, 안양시, 광명시, 군포시, 하남시

**전입 계획 시**

1.  전입 예정일 확인
2.  전입 전 포인트 전액 사용
3.  전입 완료 후 사용 불가
4.  미사용 포인트 소멸

## ❓ 자주 묻는 질문

**Q. 포인트가 부족하면 내 돈을 더 낼 수 있나요?**

A. 아니요, 혼합 결제는 불가능합니다. 보유 포인트 내에서만 사용 가능합니다.

**Q. 정해진 곳에서만 사용할 수 있나요?**

A. 농식품바우처 지정 매장에서만 사용 가능합니다.

**Q. 온라인 쇼핑몰에서도 쓸 수 있나요?**

A. 네, 농식품바우처 지정 매장으로 등록된 온라인 쇼핑몰에서 사용 가능합니다.
