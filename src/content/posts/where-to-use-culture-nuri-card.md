## 🏪 문화누리카드 사용처

## 🎭 문화예술 분야 사용처

### 📚 도서·음반

**서점 (오프라인·온라인)**

-   교보문고, 영풍문고, 예스24, 알라딘, 인터파크 도서
-   전자책: 교보 eBook, 리디북스, 밀리의 서재
-   음반: 교보핫트랙스, 신나라레코드

### 🎬 영화관

-   CGV, 롯데시네마, 메가박스 (전국 지점)
-   온라인·모바일 예매 가능
-   영화 관람권 구매 (팝콘·음료는 일부 제한)

### 🎭 공연·전시

**공연장**: 세종문화회관, 예술의전당, LG아트센터, 대학로 소극장, 지역 문화예술회관

**박물관·미술관**: 국립중앙박물관, 국립현대미술관, 전쟁기념관, 지역 박물관·미술관

**온라인 예매**: 인터파크 티켓, 예스24 티켓, 멜론 티켓

### 🎨 기타 문화예술

-   문화센터 강좌 (롯데·현대·신세계문화센터)
-   악기 판매점 (삼익악기, 영창뮤직)
-   문화용품 (화방, 사진관)

```
문화누리카드 앱 실행 → 가맹점 찾기 → 내 위치 기반 검색 → 카테고리별 필터링
```

### 💻 홈페이지

```
문화누리카드 홈페이지 → 가맹점 찾기 → 지역·업종 선택 → 가맹점 목록 확인
```

## 🚫 사용 불가 업종

**❌ 사용할 수 없는 곳**

-   마트·슈퍼마켓·편의점 (식료품 구매)
-   일반 음식점·카페·주점 (외식비)
-   병원·약국·안경점
-   대중교통·택시·주유소
-   은행·보험·통신요금·공과금
-   입시·보습학원 (문화예술 학원 제외)
-   의류·잡화·생활용품점

## 🌐 온라인 사용 시 주의사항

### ⚠️ 인터넷 사용등록 필수

온라인 가맹점에서 사용하려면 **문화누리카드 홈페이지에서 ‘인터넷 사용등록’** 필수

**등록 절차**

```
NH농협카드 홈페이지 → 마이 → 안심결제 → 일반결제 → 서비스 등록
```

-   PC에서만 등록 가능
-   본인 인증 필요
-   등록 후 온라인 결제 가능

### 💳 온라인 결제 방법

```
온라인 가맹점 → 결제 수단 '문화누리카드' 선택 → 카드번호 16자리 입력 → 유효기간·CVC 입력 → 결제 완료
```

**모바일 앱 QR코드는 오프라인 전용**이므로 온라인 결제 시 실물카드 번호 입력 필요

## ❓ 자주 묻는 질문

### ❓ 모든 영화관에서 사용할 수 있나요?

대부분의 멀티플렉스(CGV, 롯데시네마, 메가박스)에서 사용 가능하지만, 일부 소규모 영화관은 가맹점이 아닐 수 있습니다. 방문 전 확인 필요합니다.

### ❓ 편의점이나 마트에서는 왜 사용할 수 없나요?

문화누리카드는 문화·관광·체육 활동 지원 목적이므로, 식료품이나 생활용품 구매에는 사용할 수 없습니다.

### ❓ 카페나 레스토랑에서 사용할 수 있나요?

일반 음식점, 카페, 레스토랑에서는 사용할 수 없습니다. 외식비가 아닌 문화생활비 지원 목적입니다.

### ❓ 가맹점에서 사용이 거부되면 어떻게 하나요?

가맹점임에도 사용이 거부되는 경우, 문화누리카드 고객센터(1544-3412)로 문의하세요.

## 📞 전화 문의

-   문화누리카드 고객센터: 1544-3412 (평일 09:00~18:00)
