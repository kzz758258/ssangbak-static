농식품바우처 카드를 분실하셨나요?

걱정하지 마세요.즉시 행정복지센터나 고객지원센터에 분실 신고를 하면 카드 사용이 정지되고, 재발급 신청을 통해 새 카드를 받을 수 있습니다.

기존 포인트는 새 카드로 자동 이전되어 손실 없이 계속 사용 가능합니다.

재발급 기간은 약 5~7일 소요되며, 우편으로 배송됩니다. 지금부터 분실 신고 방법부터 재발급 절차까지 상세히 안내해드리겠습니다.

## 🚨 카드 분실 시 즉시 조치

### 분실 신고 (최우선)

**주소지 가족 읍·면·동 행정복지센터 또는 농식품 바우처 홈페이지와 고객지원센터(1551-0857)를 통해 분실신고 및 재발급 신청이 가능합니다.**

**분실 신고 방법**

**방법 1: 홈페이지**

-   [www.foodvoucher.go.kr](https://www.foodvoucher.go.kr/view/fma/ordmain/main)
-   고객지원 메뉴
-   분실신고 및 재발급

**방법 2: 행정복지센터 방문**

-   주소지 가족 읍·면·동 행정복지센터 방문
-   신분증 지참
-   분실 신고 접수

**방법 3: 고객지원센터 전화**

-   전화번호: 1551-0857
-   24시간 분실 신고 가능
-   전화로 즉시 신고

## 🔄 카드 재발급 신청

### 재발급 신청 방법

**최초 신청 후 수령정보를 분실한경우는 행정복지센터로 문의 필요**

**방법 1: 홈페이지**

-   [www.foodvoucher.go.kr](https://www.foodvoucher.go.kr/view/fma/ordmain/main)
-   고객지원 메뉴
-   분실신고 및 재발급

**방법 2: 행정복지센터 방문**

**방문 장소**

-   주소지 가족 읍·면·동 행정복지센터
-   신분증 필수 지참

**신청 절차**

1.  행정복지센터 방문
2.  담당 부서 안내
3.  재발급 신청서 작성
4.  신분증 제시
5.  신청 완료

**방법 3: 고객지원센터 전화**

**전화 신청**

-   전화번호: 1551-0857
-   분실 신고와 동시 신청 가능

**신청 절차**

1.  고객센터 전화
2.  ‘카드 재발급’ 요청
3.  본인 확인
4.  재발급 신청 완료

### 재발급 구비서류

**필수 서류**

-   신분증 (주민등록증, 운전면허증 등)
-   재발급 신청서 (현장 작성 가능)

**본인 확인 정보**

-   이름
-   생년월일
-   주소
-   연락처

## 📬 재발급 카드 수령

### 배송 안내

**우편 배송**

-   배송 방식: 등기우편
-   배송 기간: 신청 후 약 5~7일 소요

**배송 정보 확인**

-   농식품바우처 홈페이지 접속
-   로그인 후 수령 정보 확인
-   배송 조회 가능

### 카드 수령 후 확인

**수령 확인 사항**

1.  봉투 개봉
2.  카드 본인 명의 확인
3.  카드 번호 확인 (새 번호)
4.  안내문 확인

**새 카드 정보**

-   새로운 카드 번호 발급
-   기존 카드와 번호 상이
-   유효기간 확인

## ⚡ 재발급 카드 활성화

### 자동 활성화

**카드 수령 후 다음날 자동 활성화**

-   별도 등록 절차 불필요
-   수령일 다음날부터 사용 가능

### 즉시 활성화 (ARS 등록)

**즉시 사용을 원하는 경우**

-   고객센터 ARS(1551-0857) 이용
-   카드 사용 등록

**ARS 등록 방법**

1.  1551-0857 전화
2.  ‘카드 활성화’ 메뉴 선택
3.  새 카드 번호 입력 (16자리)
4.  생년월일 입력
5.  등록 완료 및 즉시 사용 가능

## 💰 포인트 이전

### 자동 이전

**기존 포인트 자동 이전**

-   분실 카드의 잔여 포인트
-   새 카드로 자동 이전
-   별도 신청 불필요
-   손실 없음

**이전 확인**

-   새 카드 활성화 후 잔액 조회
-   ARS 또는 가맹점 결제 시 확인
-   기존 포인트 그대로 유지

### 포인트 확인 방법

**ARS 조회**

-   1551-0857 전화
-   새 카드 번호로 잔액 조회

**가맹점 결제 시**

-   첫 결제 후 영수증 확인
-   잔여 포인트 표시

## 💡 카드 훼손 시

### 카드 교체

**훼손으로 사용 불가 시**

-   카드가 휘어짐
-   긁힘으로 인식 불가
-   마그네틱 손상

**재발급 신청**

1.  행정복지센터 방문 또는 고객센터 전화 (1551-0857)
2.  카드 훼손 사유 설명
3.  재발급 신청
4.  훼손된 카드 폐기 안내
5.  새 카드 수령

**재발급 절차**

-   분실 재발급과 동일
-   약 5~7일 소요
-   포인트 자동 이전

**분실 신고 후**

-   기존 카드는 영구 사용 정지
-   찾아도 사용 불가

**월별 사용 기한 주의**

-   당월 90% 이상 사용 필요 (이월 위해)
-   재발급 기간 고려하여 조기 신청 권장

## ❓ 자주 묻는 질문

**Q. 수령 정보를 잊어버렸어요.**

A. 행정복지센터로 문의하거나 농식품바우처 홈페이지에서 확인할 수 있습니다.

**Q. 분실 신고 후 카드를 찾았어요. 사용할 수 있나요?**

A. 아니요, 분실 신고된 카드는 영구 사용 정지되어 찾아도 사용할 수 없습니다.

재발급 받은 새 카드만 사용 가능합니다.

**Q. 카드가 휘어져서 사용이 안 돼요.**

A. 카드 훼손으로 인한 재발급도 가능합니다.

행정복지센터 방문 또는 고객센터로 연락하여 재발급 신청하세요.
