## 📝 평생교육이용권 사용 절차

### 전체 사용 흐름

**1단계: 선정 및 카드 발급**

-   평생교육이용권 선정 결과 문자 수신
-   NH농협 채움카드 발급 또는 기존 카드 확인
-   35만 원 포인트 자동 충전 확인

**2단계: 사용기관 및 강좌 검색**

-   [평생교육이용권 홈페이지](https://www.lllcard.kr/main) 접속
-   본인 지자체 선택
-   ‘사용기관/강좌’ 검색 메뉴 이용
-   원하는 교육기관 및 강좌 선택

**3단계: 강좌 수강 신청**

-   선택한 교육기관에 수강 신청
-   평생교육이용권 사용 의사 전달
-   수강 신청서 작성

**4단계: 결제**

-   NH농협 채움카드로 결제
-   카드 포인트 자동 차감
-   영수증 확인

**5단계: 교육 수강**

-   해당 강좌 수강
-   출석률 관리 (일부 기관 80% 이상 필수)

## 💻 온라인 vs 오프라인 결제 방법

### 온라인 교육기관 결제

**온라인 결제 절차**

1.  **교육기관 홈페이지 접속**
    -   평생교육이용권 사용기관 홈페이지 방문
    -   회원가입 및 로그인
2.  **강좌 선택 및 장바구니 담기**
    -   원하는 강좌 선택
    -   수강 신청하기 클릭
3.  **결제 방법 선택**
    -   결제 수단에서 ‘평생교육이용권’ 또는 ‘NH농협카드 포인트’ 선택
    -   일부 사이트는 ‘카드 결제’ 선택 후 포인트 사용
4.  **카드 정보 입력**
    -   NH농협 채움카드 번호 입력
    -   비밀번호 입력
    -   포인트 사용 금액 확인
5.  **결제 완료**
    -   잔여 포인트 확인
    -   수강 확정 메시지 확인

**초과 금액 발생 시**

-   35만 원을 초과하는 강좌의 경우
-   NH농협 채움카드 연계 계좌에 초과 금액을 미리 입금
-   또는 다른 결제 수단으로 차액 결제 가능

### 오프라인 교육기관 결제

**오프라인 결제 절차**

1.  **교육기관 방문**
    -   평생교육이용권 사용기관 직접 방문
    -   수강 신청서 작성
2.  **이용권 사용 의사 전달**
    -   평생교육이용권으로 결제할 것임을 직원에게 알림
    -   NH농협 채움카드 제시
3.  **카드 결제**
    -   카드 단말기에 카드 삽입 또는 태그
    -   비밀번호 입력
    -   포인트 결제 선택
4.  **영수증 수령**
    -   결제 내역 확인
    -   사용 금액 및 잔액 확인

## 🔍 잔액 조회 방법

평생교육이용권 잔액은 여러 방법으로 확인할 수 있습니다.

평생교육이용권 잔액은 평생교육이용권 홈페이지와 NH농협카드 홈페이지에서 조회 및 확인할 수 있습니다.  
또한 이용권 사용 시, 사용 내역이 문자메시지를 통해 안내됩니다.  
1) NH농협카드 홈페이지(https://card.nonghyup.com)에서 로그인 후, 이용권 잔액 조회  
\* NH농협카드 – 상단 메뉴 ‘공공바우처’ 클릭 ‘ – 정부지원/바우처 – 이용내역 조회

### 방법 1: 평생교육이용권 홈페이지

**조회 절차**

1.  [평생교육이용권 홈페이지](https://www.lllcard.kr/main) 접속
2.  본인 지자체 지역 선택 후 로그인
3.  ‘마이페이지’ 또는 ‘이용권 조회’ 메뉴 클릭
4.  현재 잔액 확인

### 방법 2: NH농협카드 홈페이지

**조회 절차**

1.  NH농협카드 [홈페이지 접속](https://card.nonghyup.com/servlet/IpCc2021R.act)
2.  로그인
3.  ‘공공바우처’ 선택
4.  ‘정부지원/바우처’ 선택
5.  이용내역 조회
6.  평생교육이용권 포인트 잔액 확인

### 방법 3: 결제 시 자동 확인

**자동 조회**

-   온라인/오프라인 결제 시 영수증에 잔액 표시
-   이용권 사용 시 사용 내역이 문자메세지로 안내

### 방법 4: 고객센터 전화 문의

**전화 조회**

-   거주지 지역 평생교육이용권 상담센터 전화
-   본인 인증 후 잔액 조회 요청

-   사용 마감: 2025년 12월 31일
-   2026년 1월 개강 강좌: 신청 불가
-   2026년으로 이월: 불가능

### 본인 사용 원칙

**중요 규정**

-   ✅ 이용권은 반드시 본인만 사용 가능
-   ❌ 가족(배우자, 자녀 포함)에게 양도 불가
-   ❌ 타인에게 판매, 대여, 양도 불가
-   ❌ 대리 수강 불가

**위반 시 조치**

-   이용권 환수 조치
-   향후 평생교육이용권 제도 참여 제한
-   평생교육법에 따른 법적 조치 가능

### 교재비 사용 규정

**사용 가능한 경우**

-   강좌를 수강하는 사용기관에서 구매하는 교재
-   해당 강좌 수강에 직접 필요한 교재

**사용 불가능한 경우**

-   강좌 수강 없이 교재만 단독 구매
-   외부 서점이나 온라인 서점에서 구매
-   E-Book, PDF 등 전자교재 형태

## 💰 환불 및 중도 해지

### 환불 규정

**미사용 금액**

-   사용 기간 종료 시 미사용액은 자동 소멸
-   현금 환불 불가
-   다음 연도 이월 불가

**강좌 환불**

-   강좌별 환불 규정은 각 교육기관의 규정을 따름
-   일반적으로 개강 전: 전액 환불 가능
-   개강 후: 수강 일수에 따라 부분 환불

**환불 금액 처리**

-   환불된 금액은 평생교육이용권 포인트로 재충전
-   사용 기간 내에 다른 강좌에 재사용 가능

### 중도 포기

**주의사항**

-   강좌를 중도 포기해도 사용한 금액은 환불 불가 (교육기관 규정에 따름)
-   신중하게 강좌를 선택할 것을 권장

**마감 전 체크리스트**

-   11월 말: 잔액 확인
-   12월 초: 사용 계획 수립
-   12월 중순: 마지막 강좌 신청
-   12월 말: 전액 사용 확인

## ❓ 자주 묻는 질문

**Q. NH농협 채움카드가 없는데 어떻게 하나요?**

A. NH농협은행 지점을 방문하거나 온라인으로 카드를 발급받으시면 됩니다. 평생교육이용권 선정 후 발급해도 늦지 않습니다.

**Q. 35만 원을 넘는 강좌를 수강하고 싶은데 가능한가요?**

A. 네, 가능합니다. 초과 금액은 본인이 부담하시면 됩니다. NH농협 채움카드 연계 계좌에 차액을 입금하거나 다른 결제 수단을 사용할 수 있습니다.

**Q. 온라인 결제 시 포인트 사용이 안 되는데 어떻게 하나요?**

A. 해당 교육기관 고객센터에 문의하시거나, 결제 수단 선택 시 ‘카드 결제’ 후 포인트 사용 옵션을 찾아보세요. 기관마다 결제 방식이 다를 수 있습니다.

**Q. 강좌를 취소하면 환불된 금액은 어떻게 되나요?**

A. 환불된 금액은 평생교육이용권 포인트로 재충전되어 사용 기간 내에 다른 강좌에 사용할 수 있습니다.

**Q. 12월 31일까지 다 못 쓰면 어떻게 되나요?**

A. 미사용 금액은 자동으로 소멸되며, 환불이나 이월이 불가능합니다. 반드시 기간 내에 사용하세요.

**Q. 가족이 대신 수강해도 되나요?**

A. 아니요, 절대 불가능합니다. 본인만 사용 가능하며, 위반 시 환수 조치 및 향후 참여 제한이 있습니다.

**Q. 잔액 조회는 어디서 하나요?**

A. 평생교육이용권 홈페이지, NH농협카드 앱, 결제 시 영수증 등에서 확인할 수 있습니다.
