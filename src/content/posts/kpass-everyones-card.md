## 🚦 모두의카드란?

‘모두의카드’는 **한 달 동안 환급 기준금액을 초과해 지출한 대중교통비를 전액 환급**해주는 **무제한 환급형 K패스**입니다.

그리고 모두의카드는 **새로운 카드 발급이나 신청 필요 없이 기존 K패스 카드 자동 적용** 방식입니다.

### ✔️ 기존 K패스와 다른 점

구분

기존 K패스

모두의카드

환급 방식

이용 금액의 20~53.3% 환급

기준액 초과분 전액 환급

이용 조건

월 15회 이상 이용

횟수 조건 없음

적용 지역

제한적

전국 모든 지역(시내·광역·지하철·GTX 포함)

카드 필요 여부

K패스 카드

기존 K패스 카드 그대로 사용

즉, **출퇴근·통학처럼 월 교통비가 높을수록 혜택이 커지는 구조**입니다.

## 📌 일반형 vs 플러스형

2026년 개편에서 핵심적으로 가장 중요한 부분은 모두의카드 종류 **2가지 유형 제공**입니다.  
  
💡 **두 유형 중 더 유리한 쪽으로 자동 적용**되므로, 별도 선택 없이 매월 본인에게 최적화된 환급을 받을 수 있습니다.

### 🔹 일반형

-   1회 이용요금이 _3,000원 미만_인 교통수단 이용 시 적용
-   시내버스, 마을버스, 지하철 중심 이용자에게 유리
-   일반형 환급 기준금액이 더 낮아서 **환급 받기 쉬움**

### 🔹 플러스형

-   \*\*모든 대중교통 수단(GTX·광역버스 포함)\*\*에 적용
-   출퇴근·장거리 통학 등 고비용 이용자에게 최적
-   기준액은 높지만 지출이 많은 이용자는 가장 큰 혜택

## 🌍 지역별 환급 기준 금액

모두의카드는 거주 지역에 따라 **수도권 / 일반 지방권 / 우대지원 / 특별지원 지역**으로 나뉘며 기준액이 다르게 적용됩니다.

### ✔️ 수도권

-   **일반 국민:** 일반형 6.2만원 / 플러스형 10만원
-   **청년·2자녀·어르신:** 일반형 5.5만원 / 플러스형 9만원
-   **3자녀 이상·저소득:** 일반형 4.5만원 / 플러스형 8만원

### ✔️ 일반 지방권

-   일반 국민: 5.5만원 / 9.5만원
-   청년·2자녀·어르신: 5만원 / 8.5만원
-   3자녀 이상·저소득: 4만원 / 7.5만원

### ✔️ 우대지원 지역

-   일반 국민: 5만원 / 9만원
-   청년·2자녀·어르신: 4.5만원 / 8만원
-   3자녀 이상·저소득: 3.5만원 / 7만원

### ✔️ 특별지원 지역

-   일반 국민: 4.5만원 / 8.5만원
-   청년·2자녀·어르신: 4만원 / 7.5만원
-   3자녀 이상·저소득: 3만원 / 6.5만원

> 교통 인프라가 부족한 지역일수록 기준액을 낮춰 **더 쉽게 환급 혜택을 받을 수 있도록 설계됨**.

## 👵 ‘어르신(65세+) 유형’ 신설

2026년부터는 \*\*기본형에서 환급률 30%\*\*가 적용되는 **어르신 전용 유형**이 생깁니다.

✔️ 기존: 모든 연령 일반 국민 20%  
✔️ 개편: 65세 이상은 **환급률 30% 적용**

이는 **교통약자 이동권 강화** 정책의 일환입니다.

## ✨ 실제 적용 환급 예시

### 🧑 사례 ① 서울 거주, 3자녀 A씨

-   월 교통비 60,000원  
    → **일반형 기준금액 45,000원**  
    → 환급: 15,000원

### 👩 사례 ② 부산 거주 일반 B씨

-   월 교통비 110,000원  
    → 기준금액 55,000원  
    → 환급: 55,000원

### 🧑‍🎓 사례 ③ 경기도 거주 청년 C씨

-   월 교통비 150,000원(GTX + 버스)  
    → 플러스형 기준금액 90,000원  
    → 환급: **60,000원**

확실히 **대중교통 이용량이 많을수록 혜택이 극대화**됩니다.

### 🔹 1단계: K패스 카드 준비

기존 K패스 사용자라면 그대로 사용 가능  
(신규 사용자는 제휴카드사에서 카드 발급 가능)

### 🔹 2단계: K패스 앱·누리집 로그인

-   K패스 앱 설치
-   회원가입 및 카드 등록
-   환급 조회 기능 자동 활성화

### 🔹 3단계: 대중교통 이용

-   별도 선택 필요 없음
-   월 이용금액 자동 집계

### 🔹 4단계: 혜택 자동 선택

시스템에서 다음 두 가지를 비교해  
가장 환급이 큰 방식으로 자동 적용합니다.

-   기존 K패스 방식
-   모두의카드(일반형/플러스형) 방식

### 🔹 5단계: 환급 수령

-   앱에서 환급액 확인
-   계좌 자동 입금
