## 🚨 **카드 분실 시 해야 할 첫 조치**

기후동행카드 분실 시 가장 먼저 해야 할 일은 **카드 사용 중지**입니다.  
  
분실된 카드로 다른 사람이 탑승하면 **실제 사용 금액이 모두 본인 요금으로 청구**될 수 있어 즉각 조치가 필요합니다.

### ✔ 분실카드 사용 중지 방법

-   모바일 티머니 앱 접속
-   로그인
-   **전체메뉴 → 기후동행카드 → 분실·도난 환불 → 분실/도난 접수**

분실 신고를 완료하면 카드가 자동으로 차단되어 더 이상 대중교통 결제가 불가능해집니다.

## 💸 **분실/도난카드 환불 방법**

분실 신고만으로 끝나지 않습니다.  
  
기후동행카드의 정액 금액(충전금)은 반드시 **사용기간 만료 전** 환불 접수를 해야 합니다.

### ✔ 환불 가능 조건

-   **사용기간 만료 전** 반드시 환불 신청해야 합니다.  
    → 만료 후에는 **어떠한 사유라도 환불 불가**  
    → 고장·분실·정상 완료 모두 포함

### ✔ 환불 금액 계산 방식

환불액은 다음과 같이 계산됩니다:

> **환불액 = 카드 충전금 – 대중교통 실사용금액 – 수수료 500원**

### ✔ 청소년 할인 이용자의 주의사항

청소년 할인 적용 상태에서 분실한 경우, **환불금액은 할인된 금액이 아닌 ‘일반권 기준 실사용금액’으로 계산**됩다.

## 📝 **기후동행카드 분실 환불 신청 절차**

![](https://ssangbak.com/wp-content/uploads/2025/12/image-3.png)

### **① 모바일 티머니 앱 접속**

-   모바일 앱에서 로그인
-   전체 메뉴로 이동

### **② 분실카드 환불 정보 입력**

경로:

> **전체메뉴 → 기후동행카드 → 분실도난환불 → 분실·도난 환불 신청**

입력 정보:

-   입금 계좌 정보 (현금으로 충전했을 시)
-   사용자 본인 정보
-   사용기간 내 환불 접수 필수

### **③ 환불 여부 심사(티머니 측 처리)**

-   신청 후 티머니에서 카드 사용내역 및 잔액 확인
-   실사용금 제외 후 환불액 확정

### **④ 환불금 입금 또는 카드 결제 취소 처리**

-   **현금 충전:** 접수 완료 후 약 5일 내 계좌 입금
-   **카드 충전:** 접수 후 5일 내 카드사 결제 취소 요청 자동 진행
-   카드사 처리까지 영업일 기준 3~7일 소요

## 🔄 **분실 카드 재사용 등록 방법**

카드를 잃어버렸는데 뒤늦게 발견한 경우, 해당 카드를 다시 사용할 수도 있습니다.

단, 조건이 있습니다:

### ✔ 재사용 등록 조건

-   환불을 신청한 분실/도난 카드의 경우  
    **사용기간 만료일 기준 1일 ~ 최대 3일 이후 재사용 가능**
-   재충전 시 사용 개시일 선택 가능
-   등록된 카드 번호 삭제 시 _‘재사용 등록 불가’_ 상태가 되니 주의

### ✔ 재사용 등록 절차

1.  **티머니 앱 접속**
2.  메뉴 이동 전체메뉴 → 기후동행카드 → 분실도난환불 → 분실도난 환불 조회 → ‘재사용 버튼’
3.  등록 후 재사용 가능 시점 확인  
    (사용기간 종료 후 1~3일 이후)

### ✔ 적용 예시

-   3월 3일~3월 30일 사용권 보유
-   3월 15일 분실/도난 환불 신청
-   3월 20일 카드 발견 후 ‘재사용 등록’  
    → **4월 2일 이후 재사용 가능**

## 🆕 **새카드 재발급 후 등록 방법**

카드를 완전히 잃어버렸다면 **새로운 실물 기후동행카드를 구매 후 등록**해야 합니다.

### ✔ 새 카드 구매처

-   서울 지하철 1~8호선 고객안전실
-   역사 내 편의점(GS25, CU, 세븐일레븐, 이마트24 등)
-   소비자 판매가: **3,000원**

### ✔ 새 카드 등록 방법

1.  카드 뒷면 QR코드 스캔
2.  티머니 홈페이지 접속 후 [새 카드 등록](https://www.t-money.co.kr/ncs/pct/cardrgt/ReadCardNoRgt.dev?set=add)
3.  **실물 카드 등록(1인 1카드 원칙)**
4.  지하철역 무인충전기 또는 모바일티머니 앱으로 충전
    -   김포골드라인은 현금 충전만 가능

### ✔ 사용 시작 전 주의

-   초기 충전 후 5일 이내 **사용개시일 선택 가능(단기권 제외)**
-   만료 전 재충전은 사용개시일 변경 불가

    추가 요금 발생 여부가 카드 사용내역에 반영되어 환불액에도 영향

## ⚠ 반드시 기억해야 할 핵심 5가지

1.  **사용기간 만료 후에는 환불 절대 불가**
2.  분실 카드 환불은 **티머니 홈페이지 회원가입 + 카드 등록 필수**
3.  카드 실사용 금액은 환불액에서 제외
4.  발견한 카드 재사용 시 **사용기간 종료 후 1~3일 뒤 가능**
5.  새 카드 등록 시 기존 카드 삭제하면 복구 불가
