## 🟩 기후동행카드 등록이 필요한 이유

기후동행카드는 단순 교통카드가 아니라 “정기권 시스템”이기 때문에 구매만으로는 혜택이 적용되지 않습니다.

### ✔ 등록이 필요한 핵심 이유

-   정기권(30일권, 단기권) 기능 활성화
-   분실 시 잔액 이전 가능
-   후불카드는 등록해야 **월 부담한도 적용** (미적용 시 전체 요금 청구)
-   따릉이 등 연계 서비스 이용 가능
-   충전 이력 및 이용내역 조회 가능

📌 **등록하지 않으면 환불·혜택·사용 모두 불가 상태**가 될 수 있으니 구매 즉시 등록이 필수입니다.

## 🟦 모바일 기후동행카드 등록방법

모바일 기후동행카드는 **모바일 티머니 앱**에서 발급과 등록이 동시에 이루어집니다.

### ✔ 모바일 카드 등록 절차

1.  **모바일 티머니 앱 다운로드**
    -   안드로이드 OS 12 이상 지원
    -   **iOS는 미지원**
2.  앱 회원가입 및 본인 인증
3.  “기후동행카드” 메뉴에서 모바일 카드 발급
4.  발급 완료 후 정기권 충전
5.  충전 후 5일 이내 사용 개시일 선택 가능  
    (단기권은 개시일 선택 불가)

### ✔ 특징 및 주의사항

-   휴대폰 NFC 활성화 필수
-   배터리 부족 시 사용 불가
-   계좌이체·본인 명의 신용/체크카드 결제 가능

모바일형은 **정기적으로 출퇴근하는 성인층·학생층**에게 유리한 형태입니다.

[👉 티머니 공식 안내 바로가기](https://tmpay.tmoney.co.kr:1443/resources/html/custom/climateServiceGuide.html)

## 🟩 실물 기후동행카드 등록 방법

실물 기후동행카드는 **구매 후 반드시 등록해야 사용 가능**합니다.

### ✔ ① 실물 카드 구매

-   서울교통공사 역사 고객안전실(1~8호선)
-   서울 지하철역 편의점(GS25, CU, 세븐일레븐, 이마트24 등)

**결제가능:** 현금·신용/체크카드 (단, 고객안전실은 현금만 가능)

### ✔ ② 실물 카드 등록

구매한 카드 뒷면 QR코드를 스캔하여  
👉 **티머니 카드&페이 홈페이지**에서 등록합니다.

#### 등록 절차

1.  카드 뒷면 QR코드 스캔
2.  티머니 카드&페이 회원가입
3.  카드 등록 (1인 1카드 원칙)
4.  등록 완료 후 충전 가능

#### 주의사항

-   **등록하지 않은 실물카드는 환불 불가**
-   등록하지 않으면 따릉이·정기권 기능 사용 불가
-   실물 카드는 법인카드 등록 불가

## 🟧 후불 기후동행카드 등록 방법

후불형 기후동행카드는 신용카드사에서 발급받은 후 반드시 **티머니 카드&페이 홈페이지에 등록**해야 합니다.

### 등록 절차

1.  제휴 카드사에서 카드 발급
2.  티머니 카드&페이 홈페이지 접속
3.  회원가입 후 보유 카드 등록
4.  등록 완료 후 월 부담한도 자동 적용

📌 **등록하지 않을 경우**  
정기권 혜택이 적용되지 않아  
👉 대중교통 이용요금 전체가 청구됩니다.

## 🟩 기후동행카드 충전 방법 정리

기후동행카드는 **선불형만 충전 가능**, 후불형은 충전 불필요합니다.

### ✔ ① 지하철역 무인충전기 충전

충전 가능한 위치는 다음과 같습니다.

#### ✔ 충전 가능 장소

-   서울지하철 1~8호선
-   2~9호선
-   우이신설선
-   신림선
-   김포골드라인(현금 충전만 가능)

#### ✔ 충전 불가 장소

-   코레일
-   공항철도
-   신분당선

#### ✔ 결제수단

-   현금
-   체크/신용카드
-   _김포골드라인은 현금 충전만 가능_

### ✔ ② 모바일 티머니 앱 충전

OS별 경로가 다릅니다.

#### AOS(안드로이드)

-   전체 메뉴 → **기후동행카드** → 실물 카드 조회/충전

#### iOS(아이폰)

-   전체 메뉴 → **티머니카드** → 실물 카드 조회/충전

> **아이폰(iOS) 사용자는 모바일 기후동행카드를 발급할 수 없다.**  
> **  
> 다만, 2025년 1월 25일부터는 아이폰에서도 실물 기후동행카드를 스마트폰 접촉 방식으로 충전·조회할 수 있도록 지원이 확대된다.**

## 🟩 사용 개시일 설정 규칙 (필독)

핵심 포인트:

### ✔ 충전 후 5일 이내 사용 개시일 설정 가능

예: 1월 1일 충전 → 1월 1~5일 중 원하는 개시일 선택 가능  
※ 단기권은 개시일 변경 불가

### ✔ 개시일 이전에는 이용 불가

정기권은 개시일 00시부터 활성화됩니다.

## 🟦 기후동행카드 재충전 규칙

### ✔ ① 30일권 재충전

#### 사용기간 만료 전

-   사용 개시일 변경 불가
-   권종 변경 불가
-   만료일 기준 +30일 자동 연장
-   충전은 **만료 5일 전부터 가능**
-   단, 오전 12~04시는 만료 전 충전 불가

#### 사용기간 만료 후

-   최초 충전과 동일하게  
    **사용 개시일·권종 선택 가능**

### ✔ ② 단기권 재충전

-   **사용기간 종료 후에만 재충전 가능**
-   개시일 변경 불가

## 🟩 기후동행카드 사용방법 정리

사용 방법은 다음과 같습니다.

### ✔ ① 지하철·버스·한강버스

-   모바일/실물 카드를 단말기에 태깅
-   NFC 기본모드 설정 필수 (모바일)
-   하차 미태깅 2회 이상 누적 시 이용 제한 가능
-   24시간 사용정지 가능 규정 존재

### ✔ ② 따릉이 이용

따릉이 이용은 별도 등록이 필요합니다.

#### 절차

1.  기후동행카드 구매
2.  **티머니 GO 앱** 다운로드 및 회원가입
3.  카드번호 등록
4.  이용기간 동안 따릉이 2시간 이용권 제공

📌 실물 카드 사용자는 반드시  
**티머니 카드&페이 홈페이지 등록 후 이용 가능**

## 🟩 등록 시 자주 발생하는 문제

### ✔ 카드 인식이 안 되는 경우

-   모바일: NFC 켜기, 배터리 확인
-   실물: 카드 뒷면 번호와 실제 카드 일치 여부 확인
-   후불: 티머니 홈페이지 등록 여부 반드시 확인

### ✔ 충전했는데 사용이 안 될 때

-   개시일이 미래 날짜인지 확인
-   만료 전날 0시~4시는 충전 반영 지연 가능
-   단기권은 개시 즉시 활성화

### ✔ 분실 시

-   모바일: 앱에서 즉시 정지
-   실물: 티머니 사이트에서 정지 후 **잔액 이전 가능**
