농식품바우처를 사용하다 보면 현재 얼마나 남았는지 확인이 필요하실 텐데요.

잔액 조회는 ARS 전화, 홈페이지 로그인, 가맹점 결제 시 영수증으로 간편하게 확인할 수 있습니다.

특히 매달 말일까지 90% 이상 사용해야 10% 미만만 이월되므로, 정기적인 잔액 확인이 매우 중요합니다.

지금부터 3가지 잔액 조회 방법을 상세히 안내해드리겠습니다.

[👉 농식품바우처 잔액조회](https://www.foodvoucher.go.kr/view/fma/ordmypg/myPage)

## 📞 ARS 전화 조회

### 24시간 자동 조회

**전화번호: 1551-0857**

**조회 방법**

1.  1551-0857 전화
2.  ARS 안내에 따라 ‘잔액 조회’ 메뉴 선택
3.  카드 번호 입력 (16자리)
4.  생년월일 입력 (6자리)
5.  현재 잔액 음성 안내

**장점**

-   24시간 언제든 이용 가능
-   별도 로그인 불필요
-   빠르고 간편

## 💻 홈페이지 조회

### 온라인 확인

**조회 방법**

1.  농식품바우처 홈페이지 접속
2.  로그인
3.  홈페이지 상단 ‘사용안내’ 클릭
4.  ‘사용 방법’ 클릭
5.  페이지 하단 ‘[카드사용한도(잔액)확인](https://www.foodvoucher.go.kr/view/fma/ordmypg/myPage)‘ 클릭
6.  현재 잔액 및 사용 내역 확인

**상세 내역 조회**

-   잔액조회 버튼 [카드사용내역 확인](https://www.foodvoucher.go.kr/view/fma/ordmypg/myPage) 클릭
-   거래 일시, 가맹점명, 사용 금액 확인
-   엑셀 다운로드 또는 PDF 저장 가능

## 🧾 영수증 확인

### 결제 시 즉시 확인

**확인 방법**

-   가맹점에서 결제 후 영수증 수령
-   영수증 하단에 잔액 정보 표시

**표시 내용**

-   결제 전 포인트
-   사용 포인트
-   결제 후 잔여 포인트
-   거래 일시

## 📊 이월 금액 계산

### 90% 사용 기준

**이월 규정**

-   당월 지원금액의 10% 미만만 이월 가능
-   10% 이상 미사용 시 소멸

**계산 방법**

-   (사용액 ÷ 충전액) × 100 = 사용률(%)
-   90% 이상이면 이월 가능

## 📲 SMS 알림

### 자동 안내

**알림 내용**

-   매월 1일 충전 시 SMS 발송
-   충전 금액 안내
-   사용 가능 금액 안내

## ❓ 자주 묻는 질문

**Q. 조회에 수수료가 드나요?**

A. 아니요, 모든 조회가 무료입니다.

**Q. 사용 내역을 출력할 수 있나요?**

A. 네, 홈페이지에서 엑셀 다운로드 또는 PDF 저장이 가능합니다.
