## 📱 K패스 애플페이

2025년 애플페이 티머니 공식 지원 이후, 아이폰 사용자도 실물 교통카드 없이 대중교통을 이용하고 K패스 환급 혜택까지 받을 수 있게 되었습니다.  
  
2026년부터는 K패스 정책이 대폭 개편되어 혜택이 더욱 커졌으며, ‘모두의 카드’ 서비스로 자동으로 유리한 혜택을 적용받을 수 있습니다.

평소 대중교통을 자주 이용하는 아이폰 유저라면 필수로 등록해야 할 K패스 애플페이, 3분 만에 연동하는 방법을 지금 바로 알려드립니다.

## 🎯 등록 전 준비사항

### 필요한 것

**필수 항목**

-   아이폰 (iOS 최신 버전 권장)
-   애플페이에 등록된 신용/체크카드
-   모바일티머니 앱
-   본인 인증 수단 (카카오뱅크, 토스, 휴대폰 번호)

**등록 소요 시간**: 약 3분

## 📲 STEP 1: 아이폰 애플페이 티머니 등록

### 지갑 앱에서 티머니 추가하기

```
아이폰 기본 '지갑' 앱 실행
    ↓
오른쪽 상단 '+' 버튼 클릭
    ↓
'교통카드' 메뉴 선택
    ↓
대한민국 항목 'Tmoney' 선택
    ↓
약관 동의
    ↓
충전 금액 입력
    ↓
애플페이 등록 카드로 결제
    ↓
티머니 카드 추가 완료
```

### 💡 꿀팁: 익스프레스 교통카드 설정

**설정 방법**

```
설정 → 지갑 및 Apple Pay → 익스프레스 교통카드 → Tmoney 선택
```

이렇게 설정하면 **페이스 ID 인증이나 잠금 해제 없이** 아이폰을 단말기에 갖다 대기만 하면 자동으로 결제됩니다.

## 📱 STEP 2: 모바일티머니 앱 설치 및 K패스 등록

### 티머니 앱 설치

```
앱스토어에서 '모바일티머니' 검색
    ↓
앱 다운로드 및 설치
    ↓
회원가입 및 로그인
    ↓
본인 인증 (카카오뱅크/토스/휴대폰)
```

### K패스 등록 (앱 내)

```
티머니 앱 실행
    ↓
왼쪽 상단 메뉴 → 'K-패스' 선택
    ↓
'K-패스 등록' 메뉴 진입
    ↓
등록할 티머니 카드 선택
    ↓
카드 번호 복사
    ↓
K패스 홈페이지 자동 이동
```

## 🌐 STEP 3: K패스 홈페이지 연동

### 홈페이지 가입 및 카드 등록

```
K패스 홈페이지 회원가입
    ↓
로그인
    ↓
'K-패스 카드가 있으신가요?' 알림 → '네' 선택
    ↓
복사한 티머니 카드 번호 붙여넣기
    ↓
'카드 유효성 체크' 클릭
    ↓
등록 완료
```

**전체 과정 완료!** 이제 대중교통 이용 시 자동으로 K패스 혜택이 적용됩니다.

## 💸 K패스 환급 방식

### T마일리지 적립 시스템

아이폰 티머니 K패스는 **계좌 입금이 아닌 T마일리지 적립 방식**으로 운영됩니다.

**적립 및 사용**

-   대중교통 이용 후 자동 적립
-   적립일로부터 5년간 유효
-   모바일티머니 앱에서 확인 가능

**현금화 방법**

```
모바일티머니 앱
    ↓
T마일리지 → 페이머니 전환
    ↓
현금 환불 신청
    ↓
계좌로 입금
```

**활용처**

-   편의점 결제
-   교통카드 충전
-   현금 환불

## ⌚ 애플워치 사용자 주의사항

### 애플워치 K패스 등록

**등록 방법**

```
티머니 앱 → K-패스 등록
    ↓
'워치 티머니' 선택
    ↓
K패스 홈페이지에서 카드 등록
```

### ⚠️ 중요 안내

**아이폰과 애플워치 동시 등록 불가**

-   공식적으로 동시 사용 미지원
-   하나의 기기만 선택하여 등록
-   설정을 옮겨가며 사용은 가능하나 권장하지 않음

## ❓ 자주 묻는 질문

### ❓ 애플페이 티머니는 충전이 필요한가요?

네, 일반 교통카드처럼 사전 충전이 필요합니다. 지갑 앱에서 언제든지 충전 가능하며, 애플페이에 등록된 카드로 결제됩니다.

### ❓ K패스 환급은 언제 받나요?

매월 대중교통 이용 후 다음 달에 T마일리지로 자동 적립됩니다. 적립된 마일리지는 앱에서 확인 가능합니다.

### ❓ 아이폰과 애플워치 둘 다 등록할 수 있나요?

공식적으로는 동시 등록을 지원하지 않습니다. 하나의 기기를 선택하여 사용하시기 바랍니다.

### ❓ 익스프레스 교통카드 설정은 필수인가요?

필수는 아니지만, 설정하면 아이폰을 단말기에 대기만 해도 자동 결제되어 매우 편리합니다.

### ❓ T마일리지 유효기간이 있나요?

네, 적립일로부터 5년간 유효합니다. 기간 내에 사용하시기 바랍니다.
